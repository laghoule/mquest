# Assembly Sprites

A collection of assembly language examples for sprite manipulation.

## Files

*   `basic.asm`: A basic sprite example.
*   `moving.asm`: An example of a moving sprite.
*   `vsync.asm`: An example of using vsync for smooth animation.

## Assets

*   `assets/mario.inc`: Sprite data for Mario.
*   `assets/testing.inc`: Sprite data for testing purposes.
